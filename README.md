# Gear Library
A cross-platform abstraction library with asynchronous I/O.

## Modules
 * gear.net
 * gear.codec
 * gear.event
 * gear.serialization

## Platforms
 * FreeBSD
 * Windows
 * macOS
 * Linux
 * Android

## Support compilers
 * D Compiler 2.099 +
 * LDC 1.29
 * GDC 12.1
