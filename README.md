# Gear Library
A refined core library for writing reliable asynchronous applications with D programming language.

## Modules
 * gear.buffer
 * gear.codec
 * gear.event
 * gear.io
 * gear.logging
 * gear.net
 * gear.serialization
 * gear.util

## Platforms
 * FreeBSD
 * Windows
 * macOS
 * Linux
 * Android

## Requirements
D Compiler 2.098+.

## Avaliable versions
| Identifier | Description | 
|--------|--------|
| GEAR_DEBUG |  Used to log some debug messages |
