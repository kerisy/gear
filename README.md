# Geario Library
A cross-platform abstraction library with asynchronous I/O.

## Modules
 * geario.net
 * geario.codec
 * geario.event
 * geario.serialization

## Platforms
 * FreeBSD
 * Windows
 * macOS
 * Linux
 * Android

## Support compilers
 * D Compiler 2.099 +
 * LDC 1.29
 * GDC 12.1
